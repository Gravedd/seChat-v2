# seChat - Chat with encryption
Цель проекта: разработать PWA приложение для обмена сообщениями, где сообщения шифруются по ключу.<br>
<b>Стек:</b> php, laravel, ratchet, mysql, javascript, vue 3(vuex, router, cli, pwa)<br>
